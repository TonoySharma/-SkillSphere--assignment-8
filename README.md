1= Project Name: SkillSphere

2=Purpose :
 SkillSphere is a modern online learning platform where users can explore courses, view details, and start their learning journey easily.
The main goal is to make skill development simple, interactive, and accessible for everyone.

3= Key Features :
-Course search functionality
- Dynamic course listing system
- Rating & course details section
- Fully responsive UI.
- Modern UI design using Tailwind CSS + HeroUI
- Massage notifications system
- Fast page loading with Next.js App Router
- Clean routing system (App Router)
- Custom 404 Not Found page
- Loading spinner during data fetching
- Swiper.js based carousel/slider
